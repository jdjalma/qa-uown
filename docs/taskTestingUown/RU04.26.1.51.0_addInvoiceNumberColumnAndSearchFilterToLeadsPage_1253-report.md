## Descrição

Adicionar nova coluna "Invoice Number" na tabela de Leads do portal Origination, posicionada imediatamente após "Term Month". O valor é obtido via `LEFT JOIN uown_los_invoice` no campo `merchant_invoice_number`. Adicionalmente, um campo de filtro "Invoice Number" é inserido no painel de filtros, permitindo busca por número de fatura via `POST /uown/los/getLeadsByCriteria` com o novo parâmetro `invoiceNumber`.

**Critérios de aceite:**
1. Coluna "Invoice Number" exibida na tabela de Leads
2. Coluna posicionada imediatamente após "Term Month"
3. Para leads com invoice, o valor é exibido corretamente (match exato com `uown_los_invoice.merchant_invoice_number`)
4. Para leads sem invoice, a coluna fica vazia
5. Campo de filtro "Invoice Number" existe no painel de filtros (placeholder: `Search by Invoice Number`)
6. Filtrar por invoice number existente retorna os leads correspondentes
7. Filtrar por invoice number inexistente retorna "There are no records to display"
