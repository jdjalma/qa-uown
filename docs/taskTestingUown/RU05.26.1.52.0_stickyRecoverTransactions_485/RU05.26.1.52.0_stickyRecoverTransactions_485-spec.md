- **Requisito:** R7

- **Steps (opção B — ativa, se possível):**

  2. Esperar violação de UNIQUE
- **Validações:**

  - **DB:** count(`dedupe_key`) = count distinct(`dedupe_key`) — nenhuma duplicata
  - **Activity log:** N/A (operação interna)
- **Recomendação:** começar pela opção A. Opção B requer autorização explícita extra do usuário (INSERT direto) — NÃO assumir.
- **Pitfalls considerados:** dedupe local mais restritivo que da Sticky (Melhoria #7) — não testar resubmit após unsuccessful aqui
