## Descricao

Valida os novos filtros do Servicing Search Result:

- Campo **MERCHANT** (React Select, name=`merchantName`)
- Campo **LOCATION** (React Select, name=`location`)
- Cross-select:
  - Selecionar merchant filtra locations disponiveis
  - Selecionar location auto-preenche merchant
- API: `POST /uown/svc/getAccountsByCriteria` suporta `merchantName` e retorna resultados consistentes
