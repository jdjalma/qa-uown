3. Assere `content[0].customerAccountDomain === 'SERVICING'`, `leadPk`, `leadStatus` populado.
4. Scan de segurança: nenhuma chave sensível no payload.
5. Segunda busca com `{ last4SSN, dob }` → mesmo lead em `content[]`.
