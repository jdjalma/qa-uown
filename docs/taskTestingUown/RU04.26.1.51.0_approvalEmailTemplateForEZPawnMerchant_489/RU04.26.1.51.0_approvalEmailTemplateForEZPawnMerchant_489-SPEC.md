## Interaction Strategy

API-only + Email IMAP + DB SELECT. There is no UI surface for this feature (the merchant portal does NOT render the approval email — it is composed server-side and delivered via SMTP). UI interaction is therefore **N/A** for this SPEC, and the "Triple Validation" matrix for this test is:

- **Payload**: `sendApplication` response (`status 200`, `appApprovalStatus === 'APPROVED'`).
- **Delivered artifact (email)**: IMAP-fetched HTML body asserted via text markers + image URL status codes.
- **DB**: `uown_email_queue.template_name`, `uown_los_lead_log` CORRESPONDENCE rows.

Every CT covers at least 2 of the 3 layers. CT-01/02/03 cover all 3.

// ── CT-01 primary ───────────────────────────────────────────────────────────
expect(response.status).toBe(200);

  `SELECT template_name FROM uown_email_queue WHERE lead_pk = $1 ORDER BY pk DESC LIMIT 1`,
  [ctx.leadPk],
);

// ── CT-01 audit log ────────────────────────────────────────────────────────
const logs = await db.getRows<{ log_type: string; note: string; created_by: string }>(
  `SELECT log_type, note, created_by FROM uown_los_lead_log
   WHERE lead_pk = $1 AND log_type = 'CORRESPONDENCE'
   ORDER BY pk DESC LIMIT 5`,
  [ctx.leadPk],
);
expect(logs.length).toBeGreaterThan(0);
expect(logs[0].note).toMatch(/ApprovalEmail/);
expect(logs[0].created_by).toBeTruthy();

// ── CT-02 conditional NO MONEY DOWN assertion ──────────────────────────────
const mcfg = await db.getSingleRow<{ charge_processing_fee_before_esign: boolean }>(
  `SELECT charge_processing_fee_before_esign FROM uown_merchant WHERE ref_merchant_code = $1`,
  [data.merchant.refCode ?? 'OL90202-0001'],

const shouldContainNoMoneyDown = !mcfg?.charge_processing_fee_before_esign;

test.skip(acct?.company !== 'KORNERSTONE',
  `[CT-03] BRAND_MISMATCH — expected KORNERSTONE, got ${acct?.company} (ENV-GAP; no UPDATE per Exception 3)`);

// ── CT-04 image HTTP ───────────────────────────────────────────────────────

  const res = await request.get(url);
  expect(res.status(), `Image should return 200: ${url}`).toBe(200);
