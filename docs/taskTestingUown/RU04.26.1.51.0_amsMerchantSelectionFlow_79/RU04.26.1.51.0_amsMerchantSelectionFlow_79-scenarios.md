## Descricao da Feature

O cartao "Edit User Merchants" na pagina de detalhes do usuario (`/users/[username]`) foi completamente substituido por um novo componente. O novo componente usa:

- **Aba "Origination"** — tab para contexto do sistema
- **React Select multi-select** — para selecionar/remover comerciantes individualmente
- **"Select All"** — opcao especial que atribui todos os comerciantes (equivale a `merchantCodes = "*"`)
- **Busca/filtro no dropdown** — digitar no combobox filtra comerciantes por nome/codigo
- **"Search for Current Merchants"** — searchbox em modo read-only para filtrar comerciantes ja atribuidos (desabilitado quando "Select All" esta ativo)
- **Tags removiveis** — cada comerciante aparece como tag com botao "x" para remover
- **Clear all (x)** — icone para limpar todos os comerciantes selecionados
- **SAVE / CANCEL** — botoes para confirmar ou cancelar edicao

## User Stories

### US-AMS-01: Visualizar comerciantes atribuidos ao usuario

**Como** administrador AMS,
**Quero** ver a lista de comerciantes atribuidos a um usuario na pagina de detalhes,
**Para** entender rapidamente quais comerciantes o usuario pode acessar.

#### Criterios de Aceite
- [ ] Card "Edit User Merchants" exibe aba "Origination"
- [ ] Usuarios com `merchantCodes = "*"` exibem badge "All"
- [ ] Usuarios com comerciantes especificos exibem badges verdes com "Nome - Codigo"
- [ ] Campo "Search for Current Merchants" esta visivel
- [ ] Campo de busca esta desabilitado quando usuario tem "Select All" (`*`)
- [ ] Campo de busca esta habilitado quando usuario tem comerciantes especificos

### US-AMS-02: Editar comerciantes do usuario via React Select

**Como** administrador AMS,
**Quero** adicionar e remover comerciantes de um usuario usando o seletor de busca,
**Para** gerenciar as permissoes de acesso do usuario de forma eficiente.

#### Criterios de Aceite
- [ ] Clicar no pencil abre o modo de edicao com React Select
- [ ] Dropdown exibe comerciantes no formato "Nome - Codigo **Codigo**"
- [ ] Digitar no combobox filtra comerciantes por nome ou codigo
- [ ] Selecionar comerciante adiciona tag ao React Select
- [ ] Clicar "x" na tag remove o comerciante
- [ ] Clicar SAVE persiste as mudancas
- [ ] Clicar CANCEL reverte as mudancas

### US-AMS-03: Usar "Select All" para atribuir todos os comerciantes

**Como** administrador AMS,
**Quero** usar a opcao "Select All" para atribuir todos os comerciantes ao usuario,
**Para** conceder acesso completo de forma rapida.

#### Criterios de Aceite
- [ ] "Select All" aparece como opcao no dropdown
- [ ] Selecionar "Select All" substitui comerciantes individuais pela tag "Select All"
- [ ] Remover "Select All" permite selecionar comerciantes individuais
- [ ] Apos SAVE, API confirma `merchantCodes = "*"`

## Notas

- **Implementacao real** difere dos requisitos originais (que mencionavam modal com tabela). A implementacao usa React Select multi-select com busca integrada no dropdown.
- **Nao ha botoes "+Add", "Delete All" separados** como descrito no issue — a funcionalidade esta integrada no React Select (dropdown = add, "x" nas tags = delete individual, clear all = delete all, "Select All" = select all).
- **Permissoes** nao foram testadas separadamente pois o pencil de edicao ja controla o acesso — se o usuario nao tem permissao, o pencil nao aparece.
- **Usuario de teste:** `test.tester` (comerciantes especificos) e `jmendes.gow` (Select All / `*`).
