## US-VERIF-01: Template profissional do email de verificacao

**Como** cliente da plataforma,
**Quero** receber um email de verificacao profissional e alinhado com a marca,
**Para** sentir confianca ao acessar o Customer Portal.

### Criterios de Aceite
- [ ] Email de verificacao usa o novo template (layout moderno com branding)
- [ ] Codigo de verificacao (6 digitos) exibido claramente em caixas individuais
- [ ] Informacoes de contato presentes no email
- [ ] Links de redes sociais presentes no email
- [ ] Template UOWN usa cores azuis e dados de contato UOWN
- [ ] Template KS usa cores verdes e dados de contato Kornerstone
- [ ] Codigo funciona para completar o login no portal

## Cenarios de Teste

### CT-01: Email de verificacao UOWN - estrutura do template

**Tipo:** Hybrid (E2E + Email)
**Portal:** Website (Customer Portal)
**Pre-condicao:** Conta de cliente UOWN existente com email acessivel

**Passos:**
1. Acessar o Customer Portal (website URL)
2. Inserir email de um cliente UOWN existente
3. Submeter o formulario de login
4. Buscar email de verificacao via IMAP (`getEmailContent`)
5. Validar que o corpo HTML contem elementos de branding UOWN

**Resultado esperado:**
- Email recebido com subject contendo "Verification"
- HTML contem logo UOWN (imagem com "uown" no src)
- HTML contem icone de cadeado (lock icon)
- HTML contem titulo "Verification Code"
- HTML contem codigo de 6 digitos
- HTML contem informacoes de contato: telefone (877) 357-5474
- HTML contem links de redes sociais (Facebook, Instagram, Twitter)
- HTML contem referencia a uownleasing.com

**Tags:** @regression @critical

### CT-05: Formato do codigo de verificacao - unica linha para copia mobile

**Tipo:** Hybrid (E2E + Email)
**Portal:** Website (Customer Portal)
**Pre-condicao:** Conta de cliente existente com email acessivel

**Passos:**
1. Triggar email de verificacao (via portal login)
2. Buscar HTML do email via IMAP
3. Validar que o codigo esta em um bloco copiavel (hidden div com codigo completo)
4. Validar que o codigo esta exibido como 6 digitos individuais em caixas

**Resultado esperado:**
- HTML contem div oculto com `user-select:all` contendo o codigo completo (para copia mobile)
- HTML contem 6 celulas de tabela individuais (th:each com sequence 0-5)
- Codigo tem exatamente 6 digitos numericos

**Tags:** @regression
