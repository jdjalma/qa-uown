# Prompt Journal — fintech-playwright

Diretório de prompts históricos usados para construir fluxos complexos do projeto.

## Propósito

Guardar prompts que produziram resultados de alta qualidade para:
- Referência futura ao implementar fluxos similares
- Onboarding de novos membros
- Reprodução de padrões bem-sucedidos

## Convenção de nomes

```
{número}-{módulo}-{ação}.md
```

Exemplos:
- `01-tire-agent-unified-flow.md`
- `02-paypair-portal-migration.md`
- `03-dry-audit-refactor.md`

## Como usar

Ao enfrentar um fluxo similar a um prompt existente, ler o prompt como referência de abordagem — não copiar literalmente, mas absorver o padrão.
